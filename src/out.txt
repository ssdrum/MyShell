# MANUAL: myshell

## NAME
        myshell

## SYNOPSIS
        myshell [file]

## DESCRIPTION
        Myshell is a UNIX command interpreter that executes commands from standard input or from a file.

## ARGUMENTS 
        The argument file is assumed to be a text file containing shell commands. Myshell reads, prints, and executes the commands.

## INVOCATION 
        Myshell starts in interactive mode if no optional file is provided as command-line argument.
